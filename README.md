# Agent N8N

Zautomatyzowany agent stworzony na platformie n8n (low-code), mający na celu usprawnienie codziennych zadań związanych z kalendarzem i komunikacją.

## Co potrafi?

Kalendarz: Szybkie dodawanie i sprawdzanie wydarzeń na dany dzień/okres.

Komunikacja: Wysyłanie e-maili.

Info: Sprawdzanie aktualnej daty i godziny.

Technologia: n8n, Google API, Telegram.
